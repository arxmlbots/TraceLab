# TraceLab

> **Open-source PCB design editor for Android. Design PCBs anywhere. 🚀**

TraceLab is an open-source Android application that aims to make professional PCB (Printed Circuit Board) design accessible directly from mobile devices.

Our long-term goal is to provide a modern, offline-first PCB design experience for students, hobbyists, makers, and engineers who don't always have access to a desktop or laptop.

> **⚠️ Project Status:** Early Development (Pre-Alpha)

---

## Vision

To build a professional PCB design tool for Android that allows users to create, edit, and export PCB layouts entirely on mobile devices.

---

## Planned Features

- Infinite canvas
- Grid & snap system
- PCB layout editor
- Multi-layer PCB support
- Footprint library
- Symbol library
- Gerber export
- Drill file export
- Design Rule Check (DRC)
- BOM generation
- Plugin system
- Dark mode
- Offline-first workflow
