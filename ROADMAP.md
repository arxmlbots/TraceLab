# TraceLab Roadmap

> This roadmap outlines the planned development of TraceLab. Features and priorities may change as the project evolves.

---

# v0.0.1-alpha — Foundation

Goal: Build the core of the PCB editor.

- [ ] Initialize Android project
- [ ] Material 3 UI
- [ ] Infinite canvas
- [ ] Zoom & Pan gestures
- [ ] Grid rendering
- [ ] Coordinate system
- [ ] Canvas rendering engine
- [ ] Basic settings

---

# v0.0.2-alpha — Project Management

Goal: Create and manage PCB projects.

- [ ] New project
- [ ] Open project
- [ ] Save project
- [ ] Autosave
- [ ] Recent projects

---

# v0.1.0-alpha — Component Placement

Goal: Start designing PCB layouts.

- [ ] Footprint library
- [ ] Search footprints
- [ ] Place components
- [ ] Move components
- [ ] Rotate components
- [ ] Delete components
- [ ] Snap to grid

---

# v0.2.0-alpha — PCB Routing

Goal: Connect components.

- [ ] Manual routing
- [ ] Via placement
- [ ] Copper layers
- [ ] Track editing
- [ ] Undo / Redo

---

# v0.3.0-alpha — PCB Output

Goal: Export PCB manufacturing files.

- [ ] Gerber export
- [ ] Excellon drill export
- [ ] PNG export
- [ ] PDF export
- [ ] Project backup

---

# v0.4.0-beta — Design Tools

Goal: Improve editing experience.

- [ ] Design Rule Check (DRC)
- [ ] Measurement tool
- [ ] Layer manager
- [ ] Grid customization
- [ ] Theme customization

---

# v0.5.0-beta — Libraries

Goal: Expand available components.

- [ ] Built-in footprint library
- [ ] User footprint library
- [ ] Import footprints
- [ ] Export footprints

---

# v0.8.0-beta — Advanced Features

- [ ] Multi-layer PCB support
- [ ] Plugin system
- [ ] BOM generation
- [ ] Keyboard & mouse support
- [ ] Performance optimization

---

# v1.0.0 — Stable Release

Goal: First stable version of TraceLab.

- [ ] Stable PCB editor
- [ ] Reliable Gerber export
- [ ] Comprehensive footprint library
- [ ] Offline-first experience
- [ ] Documentation
- [ ] User manual

---

# Future Ideas

These ideas are not guaranteed but may be implemented in future versions.

- Schematic editor
- Autorouter
- 3D PCB Viewer
- SPICE simulation
- Cloud synchronization (optional)
- Collaboration features
- Plugin marketplace
- AI-assisted routing
